# RIGOL DHO800/900 IMAGE

[DHO924S SD image - Firmware 1.05 from 2026-01-20](https://github.com/EleDes/RIGOL_DHO800-900_IMAGE/releases/download/FW_1.05_Image/Rigol.DHO924S.FW.1.05.Image.xz)

### Normal users have no need for this - it is only for recovery purposes e.g if Scope is hard bricked or SD failed.

Decompress e.g. with 7zip and write with any image tool to a microSD with at least 32GB (e.g. Win32DiskImager).  
It is compatible with all DHO800/900 models, but if required can be downgraded to every model with the zelea tool: https://github.com/zelea2/rigol_vendor_bin  


More comfort with a batch file for selecting model: https://www.eevblog.com/forum/testgear/hacking-the-rigol-dho800900-scope/msg6210539/#msg6210539
